# Python Password Generator

## Description
This project is a simple Python-based Password Generator that creates secure random passwords based on user input. The program allows users to specify the desired password length and then generates a strong password using a combination of letters, numbers, and special characters.

The goal of this project is to demonstrate basic Python programming concepts such as user input, randomization, loops, and string manipulation.

## Features
- Generates secure random passwords
- Allows user to choose password length
- Uses letters, numbers, and symbols for stronger security
- Simple command-line interface

## Technologies Used
- Python
- Built-in Python libraries (`random`, `string`)

## How It Works
1. The user enters the desired password length.
2. The program combines letters, digits, and symbols.
3. A random selection of characters is used to generate the password.
4. The generated password is displayed to the user.

## Example
Enter password length: 12
Generated Password: A#k9@Lm2!pQ$
